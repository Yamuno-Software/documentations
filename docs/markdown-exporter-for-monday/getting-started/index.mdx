# Getting Started

![Markdown Exporter for Monday.com](https://pub-b0d9bb8609a9490e8a3d5e2e7f63af24.r2.dev/Markdown%20Exporter.jpg)

Welcome to Markdown Exporter for Monday.com! This guide will help you get started with exporting your Monday.com workspace documentation to markdown format.

## Quick Overview

Markdown Exporter for Monday.com allows you to:

- Export individual docs from your Monday.com workspace
- Export entire folders with preserved hierarchy
- Export your complete workspace documentation
- Download attachments and media files locally
- Preserve formatting, tables, lists, and code blocks
- Maintain internal links and cross-references
- Get organized ZIP archives with all content

## Key Features

### Single Doc Export
Export any individual document from your Monday.com workspace to markdown format while preserving all formatting and attachments.

### Folder Export
Export entire folders with their complete structure, maintaining the hierarchy and organization of your documentation.

### Workspace Export
Export your complete Monday.com workspace documentation in one go, perfect for backups or migrations.

### Local Attachments
All images, files, and media are downloaded and saved locally with proper references in your markdown files.
