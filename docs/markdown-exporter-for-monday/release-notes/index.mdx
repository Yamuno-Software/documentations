## Version 1.0.0 (Current)

**Release Date:** 30th October 2025

### Initial Release Features

**Core Functionality**

- ✅ Single doc export to markdown
- ✅ Folder export with hierarchy preservation
- ✅ Complete workspace export
- ✅ Local attachment download
- ✅ Internal link resolution

**Export Options**

- ✅ ZIP archive output
- ✅ Folder structure preservation
- ✅ Custom file naming
- ✅ Metadata inclusion
- ✅ Multiple format support

**Markdown Support**

- ✅ Full markdown syntax
- ✅ Tables and code blocks
- ✅ Lists and task lists
- ✅ Images and media
- ✅ Links and references

**Attachment Handling**

- ✅ Automatic download
- ✅ Local reference updates
- ✅ Organized folder structure
- ✅ Multiple file types
- ✅ Relative path linking
