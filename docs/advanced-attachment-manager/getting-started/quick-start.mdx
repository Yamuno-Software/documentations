# Quick Start Guide

This guide will walk you through installing and using Advanced Attachment Manager for Confluence.

![Access](https://marketplace.atlassian.com/product-listing/files/b4c374bb-b6dc-48b0-b20e-08e542a45013?width=900&height=600)

## Installation

1. Go to the [Atlassian Marketplace](https://marketplace.atlassian.com)
2. Search for "Advanced Attachment Manager for Confluence"
3. Click "Get app"
4. Select your Confluence instance
5. Click "Install"

## Your First Attachment Management Session

1. Open the app from the Confluence Apps menu
2. Browse attachments across your instance
3. Use filters to find specific files
4. Select files for bulk actions
5. View audit logs for tracking

## Basic Usage

### Viewing Attachments

1. Open the Attachment Manager dashboard
2. Browse all attachments in your instance
3. Use filters to narrow down results
4. Click on files to preview or download
5. View file context and usage information

### Filtering and Sorting

1. Use the filter panel to:
   - Filter by usage status (used/unused)
   - Filter by file type
   - Filter by uploader
   - Filter by space or page
   - Filter by upload date

2. Sort results by:
   - File size
   - Upload date
   - File name
   - Usage status

### Bulk Actions

1. Select multiple files using checkboxes
2. Choose an action:
   - Delete unused files
   - Download files
   - Export to CSV
3. Confirm the action
4. View results in audit logs

## Admin Setup

If you're an administrator:

1. Access the admin panel
2. Configure permission settings
3. Review audit logs
4. Set up cleanup policies
