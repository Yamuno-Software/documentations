# Known Issues & Limitations

Below is a list of current known issues and limitations for Advanced Attachment Manager for Confluence. We are actively working to resolve these and improve the experience.

## 1. Bulk Action Limits

- **Issue:** Performing bulk actions on a very large number of attachments may fail or be slow.
- **Workaround:** Perform actions in smaller batches.

## 2. Orphan Detection Accuracy

- **Issue:** Orphan detection may not catch all unused attachments in complex spaces.
- **Workaround:** Manually review attachments if needed.

## 3. Audit Log Delays

- **Issue:** Audit logs may take a few minutes to update after actions.
- **Workaround:** Refresh the page or check back later.

## 4. File Type Restrictions

- **Issue:** Some file types may not be supported for preview or download.
- **Workaround:** Download files directly from Confluence if preview fails.

---

If you encounter an issue not listed here, please contact our support team. We appreciate your feedback!
