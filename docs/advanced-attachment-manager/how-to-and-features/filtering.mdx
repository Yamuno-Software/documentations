# Filtering and Sorting

Advanced Attachment Manager provides powerful filtering and sorting capabilities to help you find and organize attachments efficiently.

## Filter Options

### Usage Status

Filter attachments by their usage status:

- Used attachments (referenced in content)
- Unused attachments (orphaned)
- All attachments

### File Type

Filter by common file types:

- Images (PNG, JPG, GIF, SVG)
- Documents (PDF, DOC, DOCX)
- Spreadsheets (XLS, XLSX)
- Presentations (PPT, PPTX)
- Other file types

### Uploader

- Filter by specific users
- Filter by user groups
- Filter by upload date range

### Space and Page

- Filter by specific spaces
- Filter by specific pages
- Filter by page hierarchy

## Sorting Options

Sort your filtered results by:

- File size (ascending/descending)
- Upload date (newest/oldest)
- File name (alphabetical)
- Usage status (used/unused)
- Last modified date

## Best Practices

1. **Start Broad**: Begin with minimal filters
2. **Narrow Down**: Add filters progressively
3. **Save Filters**: Save commonly used filter combinations
4. **Export Results**: Export filtered results for reporting
5. **Regular Cleanup**: Schedule regular filtering for unused files

## Tips and Tricks

- Use multiple filters together for precise results
- Save filter combinations for frequent tasks
- Export filtered results for offline analysis
- Use sorting to identify largest files quickly
- Combine filters with bulk actions for efficient cleanup
