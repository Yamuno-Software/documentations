# Bulk Actions

Advanced Attachment Manager allows you to perform actions on multiple attachments simultaneously, saving time and effort in managing your Confluence files.

![Bulk Actions](https://pub-b0d9bb8609a9490e8a3d5e2e7f63af24.r2.dev/attachment/Attachment%20manager%202/2.jpg)

## Available Bulk Actions

### Bulk Delete

Remove multiple unused or outdated files:

1. Select files using checkboxes
2. Click "Delete Selected"
3. Confirm the action
4. Review results in audit log

### Bulk Download

Download multiple files at once:

1. Select files to download
2. Click "Download Selected"
3. Choose download location
4. Files will be downloaded as a zip

## Selecting Files

### Individual Selection

- Click checkboxes next to files
- Use shift-click for ranges
- Use ctrl/cmd-click for multiple files

### Filter-based Selection

- Apply filters first
- Use "Select All" for filtered results
- Deselect specific files as needed

## Best Practices

1. **Backup First**: Always backup before bulk deletions
2. **Verify Selection**: Double-check selected files
3. **Start Small**: Test with a small batch first
4. **Check Permissions**: Ensure you have necessary rights
5. **Review Results**: Check audit logs after actions

## Safety Features

- Confirmation dialogs for destructive actions
- Permission checks before actions
- Audit logging of all operations
- Error handling and recovery

## Tips and Tricks

- Use filters to narrow down selections
- Combine with sorting for better organization
- Export before major cleanup operations
- Review audit logs regularly
