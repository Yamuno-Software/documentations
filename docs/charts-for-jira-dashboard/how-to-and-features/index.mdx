### Advanced Configurations

#### Multi-Project Analysis

Compare and analyze across projects:

```
1. Select multiple projects in data source
2. Choose "Project Comparison" mode
3. Group by relevant field (status, type, etc.)
4. Apply project-specific colors
```

#### Time-Based Reporting

Track changes over time:

```
1. Use JQL with date filters
2. Select line or area chart
3. Group by time period
4. Enable trend lines
```

## Best Practices

### Chart Design

‚úÖ **Do:**

- Use clear, descriptive titles
- Choose appropriate chart types for your data
- Limit data points for clarity
- Use consistent color coding

‚ùå **Don't:**

- Overcrowd charts with too much data
- Use similar colors for different categories
- Forget to label axes
- Ignore mobile responsiveness

### Data Accuracy

- Verify JQL queries before saving
- Regularly update filter criteria
- Check permissions for data access
- Validate grouped data makes sense

### Performance Optimization

- Limit the number of projects when possible
- Use specific JQL queries vs. broad ones
- Cache settings for frequently used charts
- Schedule refreshes during off-peak hours

## Troubleshooting Guide

### Common Issues

**Chart not displaying data?**

- Check project permissions
- Verify JQL syntax
- Ensure fields contain data
- Refresh browser cache

**Performance issues?**

- Reduce data scope
- Optimize JQL queries
- Limit grouping complexity
- Check network connectivity

**Visual glitches?**

- Clear browser cache
- Check browser compatibility
- Verify chart dimensions
- Update to latest version
