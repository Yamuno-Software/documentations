# Known Issues & Limitations

Below is a list of current known issues and limitations for Markdown Renderer for Confluence. We are actively working to resolve these and improve the experience.

## 1. Math Rendering Issues

- **Issue:** Some complex LaTeX or KaTeX expressions may not render as expected.
- **Workaround:** Simplify expressions or consult the supported syntax documentation.

## 2. Image and Attachment Embedding

- **Issue:** Only images and attachments hosted within Confluence are supported due to Atlassian's data policy.
- **Workaround:** Upload all images and files to Confluence before embedding.

## 3. Table Formatting

- **Issue:** Advanced markdown table features (row/col spans) may not render perfectly.
- **Workaround:** Use simple tables for best results.

## 4. Large Document Performance

- **Issue:** Rendering very large markdown documents may cause slowdowns.
- **Workaround:** Split large documents into smaller sections.

## 5. Emoji and Task List Support

- **Issue:** Some GitHub-style emojis or task lists may not display identically to GitHub.
- **Workaround:** Use standard markdown for best compatibility.

---

If you encounter an issue not listed here, please contact our support team. We appreciate your feedback!
