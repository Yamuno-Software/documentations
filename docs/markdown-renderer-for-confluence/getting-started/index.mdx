# Getting Started

Welcome to Markdown Renderer for Confluence! This guide will help you get started with writing and rendering markdown content directly in Confluence.

![Banner](https://marketplace.atlassian.com/product-listing/files/5992cefd-9005-49ab-83b0-8f0b54d70986)

## Quick Overview

Markdown Renderer for Confluence allows you to:

- Write and preview markdown content in real-time
- Use GitHub-Flavored Markdown syntax
- Include math formulas and code blocks
- Integrate with Confluence attachments
- Create reusable templates
