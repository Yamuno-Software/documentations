# Quick Start Guide

This guide will walk you through installing and using Markdown Renderer for Confluence.

## Installation

1. Go to the [Atlassian Marketplace](https://marketplace.atlassian.com)
2. Search for "Markdown Renderer for Confluence"
3. Click "Get app"
4. Select your Confluence instance
5. Click "Install"

## Your First Markdown Page

1. Navigate to any Confluence page
2. Start typing '/Markdown'
3. Select "Markdown Renderer"
4. Start writing markdown in the left pane
5. Preview your content in real-time on the right
6. Click "Save" when ready

![Markdown Editor Preview](https://marketplace.atlassian.com/product-listing/files/2d0bf05e-903f-4817-a527-dfb189209e64)

## Basic Usage

### Writing Markdown

1. Type your content using markdown syntax
2. Use the toolbar for common formatting
3. Preview changes in real-time
4. Save your work regularly

### Adding Attachments

1. Click the attachment icon in the toolbar
2. Select files from your Confluence instance
3. Insert them into your markdown
4. Preview to ensure proper display

### Using Templates

1. Click the template icon
2. Choose from available templates
3. Fill in the template content
4. Save your new page
