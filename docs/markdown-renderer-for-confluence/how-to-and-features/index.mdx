# How To Use & Features

Markdown Renderer for Confluence provides seamless markdown rendering capabilities within your Confluence pages.

## Live Preview

Preview markdown content in real-time:

1. Navigate to any Confluence page
2. Click the ••• (More actions) menu
3. Select "Markdown Renderer"
4. Start typing markdown content
5. See the preview update instantly

## Syntax Support

### Basic Markdown

- Headers (#, ##, ###)
- Lists (ordered and unordered)
- Links and images
- Bold and italic text
- Blockquotes

### Extended Features

- Code blocks with syntax highlighting
- Tables with alignment
- Task lists
- Horizontal rules
- Strikethrough text

## Custom Styling

- Custom CSS support
- Theme customization
- Responsive design
- Dark mode support

## Integration Features

- Confluence macro support
- Page include support
- Dynamic content rendering
- Custom macros

## Best Practices

1. **Test First**: Preview your markdown before saving
2. **Use Shortcuts**: Learn markdown shortcuts for efficiency
3. **Check Links**: Verify all links work correctly
4. **Review Formatting**: Ensure proper rendering of complex elements
5. **Backup Content**: Keep a backup of your markdown source

## Troubleshooting

If you encounter any issues:

1. Check the [Known Issues](/markdown-renderer-for-confluence/known-issues) page
2. Verify your markdown syntax
3. Check for conflicting macros
4. Ensure proper permissions
5. Contact [support](https://yamuno.atlassian.net/servicedesk/customer/portal/2/group/2/create/51) if needed
