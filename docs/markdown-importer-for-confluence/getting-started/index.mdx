# Getting Started

Welcome to Markdown Exporter & Importer for Confluence! This guide will help you get started with importing markdown files to Confluence and exporting Confluence pages to markdown.

![Banner](https://pub-b0d9bb8609a9490e8a3d5e2e7f63af24.r2.dev/importer/banner.jpg)

## Quick Overview

Markdown Exporter & Importer for Confluence allows you to:

**Import Features:**

- Import markdown files directly into Confluence (single, multiple, or ZIP)
- Preserve file hierarchy as a Confluence page tree
- Handle local images and attachments automatically
- Support full markdown syntax including code blocks, tables, and task lists
- Perform unlimited bulk imports
- Access from page level or globally (including space-wide imports)

**Export Features:**

- Export Confluence pages to markdown format
- Export single pages or entire page trees
- Export complete Confluence spaces
- Download all attachments locally with proper references
- Preserve page hierarchy in folder structure
- Get organized ZIP files with all content
