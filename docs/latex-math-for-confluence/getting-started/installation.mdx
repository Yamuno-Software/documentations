# Installation Guide

Learn how to install and configure LaTeX Math for Confluence on your Atlassian Confluence Cloud instance.

## System Requirements

### Confluence Version
- **Confluence Cloud** (latest version recommended)
- Not available for Confluence Server or Data Center

### Browser Support
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

### Permissions Required
- **Confluence Administrator** access for installation
- **Space Admin** or **Page Editor** for using the macros

---

## Installation Steps

### 1. Access Atlassian Marketplace

1. Navigate to [Atlassian Marketplace](https://marketplace.atlassian.com/)
2. Search for **"LaTeX Math for Confluence"**
3. Click on the app listing

### 2. Install the App

**Option A: From Marketplace**
1. Click **"Try it free"** or **"Buy now"**
2. Select your Confluence site
3. Authorize the installation
4. Wait for installation to complete

**Option B: From Confluence Admin**
1. Go to **Confluence Settings** ‚Üí **Manage Apps**
2. Click **"Find new apps"**
3. Search for **"LaTeX Math for Confluence"**
4. Click **"Install"**

### 3. Verify Installation

1. Go to **Confluence Settings** ‚Üí **Manage Apps**
2. Find **LaTeX Math for Confluence** in the app list
3. Verify the status shows **"Enabled"**

---

## Initial Configuration

### Access the Settings Page

1. From **Manage Apps**, find **LaTeX Math for Confluence**
2. Click **"Configure"**
3. You'll be redirected to the app settings page

### Configure Basic Settings

#### Permissions (Recommended)
1. Navigate to the **Permissions** tab
2. Choose your access control mode:
   - **Allow All Users** (default)
   - **Restrict to Whitelist**
3. If restricting, add authorized users
4. Click **"Save Changes"**

#### Review Usage Statistics
1. Navigate to the **Usage** tab
2. View current macro usage across your instance
3. Monitor which pages contain LaTeX macros

---

## Post-Installation

### Test the Installation

1. Create a new Confluence page or edit an existing one
2. Type `/latex` in the editor
3. Select **"LaTeX Math for Confluence"** from the dropdown
4. Enter a simple equation:
   ```latex
   E = mc^2
   ```
5. Save and verify the equation renders correctly

### Notify Your Team

After installation, inform your team:

## Troubleshooting Installation

### App Not Showing in Manage Apps

**Solution:**
1. Wait 5-10 minutes for installation to complete
2. Refresh the page
3. Clear browser cache
4. Contact support if issue persists

### Cannot Configure App

**Solution:**
1. Verify you have **Confluence Administrator** permissions
2. Check if the app is enabled in **Manage Apps**
3. Try accessing settings directly via the app configuration URL

### Macros Not Appearing

**Solution:**
1. Verify the app is **Enabled** in Manage Apps
2. Hard refresh the page (Ctrl+Shift+R or Cmd+Shift+R)
3. Try in a different browser
4. Check browser console for errors

## Upgrading

### Automatic Updates

LaTeX Math for Confluence updates automatically when new versions are released to the Atlassian Marketplace.

### Checking Your Version

1. Go to **Manage Apps**
2. Find **LaTeX Math for Confluence**
3. View the version number
4. Check for available updates

### Manual Update

1. Go to **Manage Apps**
2. Find the app and click **"Update"** if available
3. Wait for the update to complete
4. Refresh any open pages
