# Usage Statistics

View which pages and spaces are using LaTeX macros in your Confluence instance.

## Overview

Usage Statistics shows you where LaTeX Math macros are being used across your Confluence instance. This helps you track adoption and find pages with mathematical equations.

### Pages Using LaTeX Macros

**Page Information:**

## Troubleshooting

### No Pages Listed

**Problem:** Usage statistics shows empty or no results

**Possible causes:**
1. No LaTeX macros have been added yet
2. Macros were recently deleted
3. App needs to refresh

**Solutions:**
1. Add LaTeX macros to some pages
2. Wait a few minutes for the list to update
3. Refresh the configuration page
4. Contact support if issue persists

### Can't View Page

**Problem:** "View Page" button doesn't work

**Possible causes:**
1. Page was deleted
2. You don't have view permission
3. Space permissions changed

**Solutions:**
1. Check if page still exists
2. Verify you have space access
3. Ask admin to grant permissions
4. The page may have been archived

### Statistics Not Updating

**Problem:** New pages with macros don't appear

**Solutions:**
1. **Refresh the page** - Reload the config page
2. **Wait a moment** - Updates may take a minute
3. **Verify macro was saved** - Check the page has a LaTeX macro
4. **Contact support** - If updates don't appear after 5 minutes
