# Configuration

Configure LaTeX Math for Confluence to meet your organization's needs with comprehensive settings for access control, permissions, and usage tracking.

![Configuration Options](https://pub-b0d9bb8609a9490e8a3d5e2e7f63af24.r2.dev/latex/latex/highlights/2.jpg)

## Available Configuration Options

### [Access Control & Permissions](/docs/latex-math-for-confluence/configuration/access-control)
Control who can edit and add LaTeX macros with a simple whitelist feature. Configure access restrictions and manage the user whitelist.

### [Usage Statistics](/docs/latex-math-for-confluence/configuration/usage-statistics)
Track and monitor LaTeX macro usage across your Confluence instance to understand adoption and usage patterns.
