# Features

Discover the powerful features of LaTeX Math for Confluence that make mathematical equation editing seamless and professional.

![LaTeX Math for Confluence Features](https://pub-b0d9bb8609a9490e8a3d5e2e7f63af24.r2.dev/latex/latex/highlights/1.jpg)

## Core Features

### [Adding Macros](/docs/latex-math-for-confluence/features/adding-macros)
Learn how to add LaTeX Math macros to your Confluence pages using the `/latex` slash command.

### [Inline Macro](/docs/latex-math-for-confluence/features/inline-macro)
Embed mathematical equations directly within text for inline expressions like formulas and variables.

### [Block Macro](/docs/latex-math-for-confluence/features/block-macro)
Display standalone mathematical equations with proper spacing and formatting for complex expressions.

### [Live Preview](/docs/latex-math-for-confluence/features/live-preview)
See your equations render in real-time as you type, ensuring accuracy before publishing.

### [Dark Mode Support](/docs/latex-math-for-confluence/features/dark-mode)
Automatic theme detection that renders equations beautifully in both light and dark modes.
